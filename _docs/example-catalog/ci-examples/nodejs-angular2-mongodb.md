This tutorial will walk you through the process of adding the following:

- Build client
- Build server
- Launch composition

## Looking around
In the root of this repository you'll find a file named `docker-compose.yml`.
Let's quickly review the contents of this file:

  `docker-compose.yml`
{% highlight yaml %}
{% raw %}
version: '3'
services:
  mongodb:
    image: mongo
    ports:
      - 28017
  server:
    image: ${{build_server}}
    environment:
      - MONGO_URI=mongodb://mongodb/exampleDb
    links:
      - mongodb
    ports:
      - 9000
  client:
    image: ${{build_client}}
    ports:
      - 3000
{% endraw %}
{% endhighlight %}
